Movies of 2013

Add new movie to list.

  • The form is grabbed using a remote PHP script (cURL --> JSONP).
  • Pressing ENTER in the title field calls the Rotten Tomatoes API and fills the matched titles into a corresponding datalist.
  • Pressing ALT+ENTER (or double-tapping the field on touch devices) autofills the form with the currently selected movie's data.
